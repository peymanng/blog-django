# blog-django
blog project with django **http://peymangol.pythonanywhere.com/**

## features
- view counter
- login with github & google
- comments
- likes
- star rating
- tags

## framework and libraries
- django
- django-allauth
- django-jazzminn
- django-star-ratings
- django-taggit
- django-recaptcha
- django-ckeditor
- Bootstrap
