# blog-django
blog project with django **http://blogtest-peymangolkar.fandogh.cloud/**

## features
- view counter
- login with github & google
- comments
- likes
- star rating
- tags
- search 

## framework and libraries used:
- django
- django-allauth
- django-jazzminn
- django-star-ratings
- django-taggit
- Bootstrap
